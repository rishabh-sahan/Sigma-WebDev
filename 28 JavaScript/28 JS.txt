DOM (Document Object Model)
    The DOM represents a document with a logical tree
    It allows us to manipulate/change webpage content (HTML elements)

Selecting elements
    getElementById -> Returns the element as an object or null (if not found)
    getElementByClassName -> Returns the element as an HTML Collection or empty collection (if not found)
    getElementByTagName -> Returns the Elements as an HTML Collection or empty collection (if not found)

Query Selectors -> Allows us to use any CSS selector

Using Properties & Methods
    innerText -> Shows the visible text  contained in a node
    textContent -> Shows all the full text
    innerHTML -> Shows the full markup

Manipulating Attributes
    obj.gatAttribute(attr)
    obj.setAttribute(attr, val)

Manipulating Style 
    Style Property
    obj.style 

    using classList
    obj.classList

    classList.add() to add new classes
    classList.remove() to remove classes
    classList.contains() to check if class exists
    classList.toggle() to toggle between add & remove

Navigation 
    parentElement
    children
    previousElementSibling / nextElementSibling

Adding Elements 
    document.createElement('p')

    appendChild(element)
    append(element) -> adds at last
    prepend(element) -> adds at first
    insertAdjacent(where, element)

Removing Elements
    removeChild(element)
    remove(element)
