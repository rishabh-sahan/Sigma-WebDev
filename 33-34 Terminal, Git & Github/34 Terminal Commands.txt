Navigation
    Inside & Outside Directories

cd = change directory
    cd folder_name

cd .. = back button

    mkdir new_folder_name;

touch = To create new file
    touch file_name

Deleting Files & Folders

rm = remove files

rmdir = removes empty folders

rm -rf = removes any folders
